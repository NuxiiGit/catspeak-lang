lexer

Responsible for the lexical analysis stage of the Catspeak compiler.

ยง CatspeakLexer

function CatspeakLexer(
  buff : id.Buffer,
  offset? : real,
  size? : real,
  keywords? : struct,
)

Responsible for tokenising the contents of a GML buffer. This can be used for syntax highlighting in a programming game which uses the Catspeak env.

NOTE: The lexer does not take ownership of this buffer, but it may mutate it so beware. Therefore you should make sure to delete the buffer once parsing is complete.

Arguments

  • buff

    The ID of the GML buffer to use.

  • offset (optional)

    The offset in the buffer to start parsing from. Defaults to 0.

  • size (optional)

    The length of the buffer input. Any characters beyond this limit will be treated as the end of the file. Defaults to infinity.

  • keywords (optional)

    A struct whose keys map to the corresponding Catspeak tokens they represent. Defaults to the vanilla set of Catspeak keywords.
