CatspeakParser

function CatspeakParser(
  lexer : struct.CatspeakLexer,
  builder : struct.CatspeakIRBuilder,
) constructor { }

Consumes tokens produced by a [CatspeakLexer] and transforms them into an abstract syntax graph representing a Catspeak program.

Arguments

  • lexer

    The lexer to consume tokens from.

  • builder

    The syntax graph builder to write data to.

ยง update

static update = function() -> function

Updates the parser by parsing a simple Catspeak expression from the supplied lexer, adding any relevant parse information to the supplied syntax graph.

Returns true if there is still more data left to parse, and false if the parser has reached the end of the file.
