CatspeakGMLCompiler

function CatspeakGMLCompiler(
  asg : struct,
  interface? : struct,
) constructor { }

Consumes an abstract syntax graph and converts it into a callable GML function.

NOTE: Do not modify the the syntax graph whilst compilation is taking place. This will cause undefined behaviour, potentially resulting in hard to discover bugs!

Arguments

  • asg

    The syntax graph to compile.

  • interface (optional)

    The native interface to use.

ยง update

static update = function() -> function

Updates the compiler by generating the code for a single term from the supplied syntax graph. Returns the result of the compilation if there are no more terms to compile, or undefined if there are still more terms left to compile.
